# Claude Code Engineering Newsletter

GitHub Pages로 배포하는 8주 뉴스레터 초안입니다.

## 배포 방법

1. GitHub에서 `s0j0s.github.io` repository를 생성합니다.
2. 이 디렉토리의 파일을 repository root에 push합니다.
3. Repository Settings -> Pages에서 `Deploy from a branch`를 선택합니다.
4. Branch는 `main`, folder는 `/root`로 설정합니다.
5. 배포 후 `https://s0j0s.github.io/`에서 확인합니다.
